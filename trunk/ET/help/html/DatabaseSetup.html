Database Setup
  • If you do not have a MySQL database set up (either on a local computer or on the network) follow the instructions given Here
  • Download and run the initial database script (et20081210_empty.sql has pre-filled data for the Eau Claire area, the et20101121_no_data.sql just has the tables with no data in them).

  • Restart the Election Tracker application and use as needed
