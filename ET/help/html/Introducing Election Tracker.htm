Election Tracker Help

Overview
  • Use Election Tracker to stay ahead of the race with up to the minute vote tallys. After setting up the program for an election, votes can be entered into the system as they are collected by the polling booths. Then reports can be generated which display the currently winning candidates in each race. The reports also display the number of reporting units in each county and what percentage of the reporting units are left to go.
Features
  • View the candidates, counties, and number of reoprting units for each race with the Election Proofing Sheet.
  • See vote tallys by contest at any time with the Quick Scan Sheet which displays each candidates vote count and percentage along with the percentage of units reporting for each contest. The candidates are listed in order of decending vote count.
  • Get a detailed view of the election with the Vote Summary Sheet which displays detailed information about each contest in the election. For each county in each contest, the number and percentage of votes for each candidate is displayed with the currently leading candidate shown first in the list. Then, the number and percentage of reporting units is displayes along with the total number of votes.
  • For instant access to county contact information at a glance, you can view the County Contact Listing which displays phone numbers and web addresses used to contact each county for the latest vote counts.
  • Use the Vote Tally Sheet to collect data from the different counties to be entered in the system.
